================================================================================
DEVVAULT - KEYBOARD SHORTCUTS REFERENCE
Panduan Lengkap Pintasan Keyboard
================================================================================

================================================================================
UMUM (GENERAL)
================================================================================

CTRL + S          Simpan halaman atau perubahan admin saat ini
CTRL + F          Fokus ke pencarian halaman yang sedang dibuka
CTRL + B          Toggle buka/tutup sidebar
CTRL + K          Buka command palette (pencarian cepat)
CTRL + /          Buka modal bantuan pintasan keyboard
ESC               Tutup modal / sidebar / pencarian / command palette

================================================================================
TEMA (THEME)
================================================================================

CTRL + SHIFT + D  Toggle antara mode gelap dan mode terang

================================================================================
NAVIGASI (NAVIGATION)
================================================================================

ALT + 1           Buka kategori HTML
ALT + 2           Buka kategori CSS
ALT + 3           Buka kategori JavaScript
ALT + 4           Buka halaman Snippets
ALT + 5           Buka halaman Pinned (Halaman Dipin)
ALT + 6           Buka halaman Settings (Pengaturan)

================================================================================
EDITOR (ADMIN ONLY)
================================================================================

CTRL + SHIFT + C  Tambah blok kode (code block) baru
CTRL + SHIFT + T  Tambah blok teks (text block) baru
CTRL + SHIFT + Q  Tambah blok kutipan (quote block) baru
CTRL + SHIFT + H  Tambah blok judul (heading block) baru
CTRL + SHIFT + L  Tambah blok tautan (link block) baru

================================================================================
ADMIN ACTIONS (ADMIN ONLY)
================================================================================

CTRL + N          Buat halaman baru (new page)
CTRL + DELETE     Hapus halaman saat ini (dengan konfirmasi)
CTRL + P          Pin / unpin halaman saat ini

================================================================================
PENGEDITAN (EDITING)
================================================================================

TAB               Indent di dalam editor kode / blok
SHIFT + TAB       Outdent di dalam editor kode / blok
CTRL + Z          Undo (batalkan perubahan terakhir)
CTRL + Y          Redo (ulangi perubahan yang dibatalkan)

================================================================================
PENCARIAN (SEARCH)
================================================================================

ENTER             Lompat ke hasil pencarian berikutnya
SHIFT + ENTER     Lompat ke hasil pencarian sebelumnya

================================================================================
CATATAN PENTING
================================================================================

- Pintasan Editor dan Admin hanya tersedia ketika login sebagai admin
- Pintasan pencarian aktif saat fokus berada di input pencarian
- CTRL + K membuka command palette yang dapat digunakan tanpa mouse
- Semua pintasan dapat dilihat lagi dengan menekan CTRL + /

================================================================================
TIPS PENGGUNAAN
================================================================================

1. Gunakan CTRL + K untuk navigasi cepat antar halaman
2. Gunakan CTRL + F untuk mencari di dalam halaman yang sedang dibuka
3. Gunakan CTRL + B untuk memperluas area baca dengan menutup sidebar
4. Gunakan CTRL + SHIFT + D untuk berganti tema sesuai preferensi
5. Admin dapat menggunakan CTRL + N untuk membuat halaman baru dengan cepat
